## 🎨 颜色方案

### 邮件配色
```
主色调:
• 渐变背景: #667eea → #764ba2 (蓝紫渐变)
• 主按钮背景: #FFFFFF (白色)
• 主按钮文字: #667eea (蓝紫色)

次色调:
• 内容背景: #f8f9fa (浅灰)
• 文字颜色: #333333 (深灰)
• 边框颜色: #667eea (蓝紫色)

强调色:
• 链接颜色: #667eea (蓝紫色)
• 标签背景: #667eea → #764ba2 (渐变)
• 数据高亮: #667eea, #764ba2
```

## 📊 数据流示意图

```
┌─────────────┐
│  定时任务    │ 每日 8:00 AM
└──────┬──────┘
       ↓
┌─────────────┐
│  收集推文    │ Twitter API
└──────┬──────┘
       ↓
┌─────────────┐
│  AI 分析     │ Claude API
└──────┬──────┘
       ↓
┌─────────────┐
│  生成日报    │ 创建 DailySummary
│             │ url_slug = "2026-02-10"
└──────┬──────┘
       ↓
┌─────────────┐
│  保存数据库  │ PostgreSQL
└──────┬──────┘
       ↓
┌─────────────┐
│  生成邮件    │ 包含详情页链接
│             │ detail_url =
│             │ "http://localhost:3000/
│             │  summary/2026-02-10"
└──────┬──────┘
       ↓
┌─────────────┐
│  发送邮件    │ Resend API
└──────┬──────┘
       ↓
┌─────────────┐
│  用户收到    │ 邮箱
└──────┬──────┘
       ↓
┌─────────────┐
│  点击链接    │ 浏览器
└──────┬──────┘
       ↓
┌─────────────┐
│  访问详情页  │ Next.js
│             │ GET /api/summaries/
│             │     slug/2026-02-10
└──────┬──────┘
       ↓
┌─────────────┐
│  显示内容    │ React 组件渲染
└─────────────┘
```
